# AI 图片引擎 - 四层设计

## 模块内部状态

```python
@dataclass
class ImageEngineState:
    """图片引擎内部状态"""
    provider: str = "openai"       # 当前 provider
    model: str = "dall-e-3"        # 当前模型
    api_key: str = ""
    api_base: str = ""
    presets_dir: str = ""           # 图片 prompt preset 目录
```

## 四层设计

| 层面 | 设计内容 |
|-----|---------|
| **数据规矩** | `ImageResult(url/prompt/media_id/width/height)`、`PresetSpec(name/kind/archetype/primary_use_case/template)`；图片宽度 ≤ 1920px，大小 ≤ 5MB |
| **数据存储** | 生成的图片临时存 `/tmp/`；provider 配置从 YAML config 加载；preset 模板存 YAML |
| **数据流转** | `prompt/preset → 选择provider → 构建请求 → 调用AI API → 接收图片 → (可选)压缩 → 上传微信素材库 → 返回media_id/URL` |
| **接口层** | `generate_image(prompt, preset, model)`、`generate_cover(article_md)`、`generate_infographic(article_md, preset)` |

## 支持的 Provider

| Provider | 推荐等级 | 说明 |
|----------|---------|------|
| Volcengine (豆包) | ⭐ 首推 | 国内直连，高质量 |
| OpenAI (DALL-E) | 通用 | 国际主流 |
| OpenRouter | 聚合 | 多模型可选 |
| Gemini | 通用 | Google 原生 |
| ModelScope | 次推 | 有免费额度 |

## 核心方法

```python
class ImageEngine:
    def __init__(self, config: dict):
        self.provider = self._init_provider(config)
        self.presets = self._load_presets()
    
    def generate(self, prompt: str, preset: str = None, 
                 size: str = "1024x1024", model: str = None) -> ImageResult:
        """通用图片生成入口"""
        if preset:
            prompt = self._render_preset(preset, prompt)
        result = self.provider.generate(prompt, model or self.model, size)
        # 自动压缩 + 可选自动上传微信
        return result
    
    def generate_cover(self, article_title: str, article_summary: str) -> ImageResult:
        """生成封面图"""
        preset = "cover-hero"
        prompt = self._render_preset(preset, 
            title=article_title, summary=article_summary)
        return self.generate(prompt, size="2560x1440")  # 16:9
    
    def generate_infographic(self, article: str, preset: str) -> ImageResult:
        """生成信息图"""
        return self.generate("", preset=preset, size="1920x1080")
```
