## 接口契约

### CLI JSON Envelope

```json
{
  "success": true,
  "code": "OK",
  "message": "Success",
  "schema_version": "v1",
  "status": "completed",
  "retryable": false,
  "data": {}
}
```

### 核心接口

```python
# 转换引擎接口
class Converter:
    def convert(self, md: str, mode: str = "api", theme: str = "default") -> ConvertResult: ...

# 排版渲染器接口
class LayoutRenderer:
    def render_block(self, block_name: str, fields: dict) -> str: ...
    def validate(self, markdown: str) -> ValidationResult: ...

# 微信适配器接口
class WeChatClient:
    def upload_image(self, file_path: str) -> str: ...  # 返回 media_id
    def create_draft(self, article: dict) -> str: ...   # 返回 draft_media_id

# 图片引擎接口
class ImageEngine:
    def generate(self, prompt: str, preset: str = None) -> ImageResult: ...
```

## 状态码设计

| 状态码 | 含义 | HTTP 类比 |
|--------|------|-----------|
| `OK` | 成功 | 200 |
| `INSPECT_COMPLETED` | 检查完成 | 200 |
| `PREVIEW_READY` | 预览就绪 | 200 |
| `CONVERT_COMPLETED` | 转换完成 | 200 |
| `DRAFT_CREATED` | 草稿创建成功 | 201 |
| `LAYOUT_VALIDATED` | 排版验证通过 | 200 |
| `ERROR` | 通用错误 | 500 |
| `CONFIG_INVALID` | 配置无效 | 400 |
| `WECHAT_TOKEN_FAILED` | 微信 token 失败 | 401 |
| `WECHAT_DRAFT_FAILED` | 微信草稿失败 | 502 |
| `IMAGE_GENERATE_FAILED` | 图片生成失败 | 502 |
| `LAYOUT_VALIDATE_HAS_ERRORS` | 排版验证失败 | 400 |
