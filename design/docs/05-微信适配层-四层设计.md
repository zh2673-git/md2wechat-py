# 微信适配层 - 四层设计

## 模块内部状态

```python
@dataclass
class WeChatState:
    """微信适配层内部状态"""
    app_id: str
    app_secret: str
    access_token: str = ""
    token_expires_at: int = 0
    base_url: str = "https://api.weixin.qq.com"
```

## 四层设计

| 层面 | 设计内容 |
|-----|---------|
| **数据规矩** | `AccessToken` 缓存结构、`DraftArticle`（title/author/digest/content/thumb_media_id/show_cover_pic）、`MaterialResult`（media_id/url） |
| **数据存储** | AccessToken 运行时内存缓存（7000秒过期）；凭证从 YAML config 或环境变量 `WECHAT_APPID`/`WECHAT_SECRET` 加载 |
| **数据流转** | `校验配置→获取token(缓存)→上传素材→创建草稿→返回media_id`；草稿数据：`convert HTML → 截取摘要(120字符) → 组装DraftArticle → POST /cgi-bin/draft/add` |
| **接口层** | `WeChatClient.get_access_token()`、`upload_material(file, type)`、`create_draft(article)` |

## 核心方法

```python
class WeChatClient:
    def __init__(self, app_id: str, app_secret: str):
        self.app_id = app_id
        self.app_secret = app_secret
        self._token = ""
        self._token_expires = 0
    
    def _get_access_token(self) -> str:
        """获取 access_token（带缓存）"""
        if self._token and time.time() < self._token_expires:
            return self._token
        resp = requests.get(
            f"{BASE_URL}/cgi-bin/token",
            params={"grant_type": "client_credential", 
                    "appid": self.app_id, "secret": self.app_secret}
        )
        data = resp.json()
        self._token = data["access_token"]
        self._token_expires = time.time() + data["expires_in"] - 200  # 提前200s过期
        return self._token
    
    def upload_image(self, file_path: str, image_type: str = "thumb") -> str:
        """上传图片到微信素材库，返回 media_id"""
        token = self._get_access_token()
        with open(file_path, "rb") as f:
            resp = requests.post(
                f"{BASE_URL}/cgi-bin/material/add_material",
                params={"access_token": token, "type": image_type},
                files={"media": f}
            )
        return resp.json()["media_id"]
    
    def create_draft(self, article: dict) -> str:
        """创建草稿，返回 draft_media_id"""
        token = self._get_access_token()
        resp = requests.post(
            f"{BASE_URL}/cgi-bin/draft/add",
            params={"access_token": token},
            json={"articles": [article]}
        )
        return resp.json()["media_id"]
```

## 约束

| 约束项 | 限制 |
|--------|------|
| 标题 | ≤ 64 字符 |
| 作者 | ≤ 16 字符 |
| 摘要 | ≤ 128 字符 |
| 正文 HTML | ≤ 20,000 字符 (微信限制) |
| 封面 | 建议 900×383px (2.35:1) |
| 图片 | ≤ 5MB，≤ 1920px |
| Token 有效期 | 7200 秒 |
