## 四层设计

| 层面 | 设计内容 |
|-----|---------|
| **数据规矩** | 每个模块一个 YAML 定义文件：`name/category/serves/fields(required+optional)/when_to_use/when_not_to_use/example`；css 变量 YAML 定义主题：`colors/typography/spacing/radius` |
| **数据存储** | 内置 43 模块 YAML 编译进 Python 包；用户自定义 `~/.config/wechat-pub/layout/`；主题存 `~/.config/wechat-pub/themes/`；4 级 override：内置→全局→项目→环境变量 |
| **数据流转** | `MD解析→提取:::block语法→模块匹配→字段校验→CSS主题注入→HTML渲染→合并到文章HTML` |
| **接口层** | 4 个命令：`layout list --serves <name>`、`layout show <name> --json`、`layout render <name> --var KEY=VAL`、`layout validate --file <file.md>` |

## 核心方法设计

### `:::block` 解析流程

```python
def parse_blocks(markdown: str) -> list[BlockAST]:
    """解析 Markdown 中的 :::block 语法，返回 AST"""
    blocks = []
    pattern = r':::\s*(?:block\s+)?(\w+)(?:\[([^\]]*)\])?\s*\n(.*?)\n:::\s*'
    for match in re.finditer(pattern, markdown, re.DOTALL):
        name, title, body = match.groups()
        if is_json_format(body.strip()):
            fields = json.loads(body)
        else:
            fields = parse_key_value(body)
        blocks.append(BlockAST(name=name, title=title, fields=fields))
    return blocks

def render_block(block: BlockAST, theme: str) -> str:
    """渲染单个 :::block 为 HTML"""
    spec = load_module_spec(block.name)
    validate_fields(spec, block.fields)
    html = apply_template(spec, block.fields, theme)
    return html
```

def inject_theme(css_vars: dict) -> str:
    """生成微信兼容的内联 CSS"""
    css = ":root {\n"
    for key, val in css_vars.items():
        css += f"  --{key}: {val};\n"
    css += "}\n"
    # 每个元素使用 CSS 变量
    return wrap_inline_style(css)

## 43 模块数据定义格式

```yaml
# layout/opening/hero.yaml 示例
schema_version: "1"
name: hero
version: "1.0.0"
since: "1.0.0"
category: opening
serves: [attention, readability]
when_to_use: 文章开篇第一屏，替代普通H1标题
when_not_to_use: 数据报告的开头用metrics；一篇文章只用一个hero
anti_pattern: |
  title 写成描述句子而不是判断
fields:
  required:
    - name: eyebrow
      description: 标签词
      example: "深度观察"
    - name: title
      description: 主标题，必须是一句判断
      example: "公众号排版的真问题"
  optional:
    - name: subtitle
      description: 副标题
    - name: cta_text
      description: 开篇钩子
example: |
  :::block hero
  eyebrow: 深度观察
  title: 公众号排版的真问题
  subtitle: 不是好不好看，是读者读不读得完
  :::
```

## 对外接口契约

```python
@dataclass
class ConvertRequest:
    markdown: str
    mode: str = "api"          # api(确定) | ai(prompt)
    theme: str = "default"
    title: str = ""
    author: str = ""
    digest: str = ""

@dataclass  
class ConvertResult:
    success: bool
    html: str = ""
    error: str = ""
    images: list[ImageRef] = None

@dataclass
class ValidationResult:
    valid: bool
    errors: list[ValidationError]
    warnings: list[str]

class LayoutEngine:
    """排版引擎对外接口"""
    def convert(self, req: ConvertRequest) -> ConvertResult: ...
    def inspect(self, md: str) -> InspectResult: ...
    def preview(self, md: str) -> str: ...
    def layout_list(self, serves: str = None, category: str = None) -> list[ModuleSpec]: ...
    def layout_show(self, name: str) -> ModuleSpec: ...
    def layout_render(self, name: str, vars: dict) -> str: ...
    def layout_validate(self, md: str) -> ValidationResult: ...
```
