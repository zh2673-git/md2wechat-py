# 完整发布流程

将 Markdown 文章从检查到推送微信草稿箱的完整链路。

## 步骤

1. 检查 CLI: `wechat-pub version`
2. 检查配置: `wechat-pub config validate`
3. 发现能力: `wechat-pub capabilities --json`
4. 检查文章: `wechat-pub inspect <article.md> --json`
5. 可选 AI 去痕: `wechat-pub humanize <article.md>`
6. 生成封面: `wechat-pub generate_cover --article <article.md>`
7. 本地预览: `wechat-pub convert <article.md> --preview`
8. 用户确认后推送草稿: `wechat-pub convert <article.md> --draft --cover cover.jpg`
