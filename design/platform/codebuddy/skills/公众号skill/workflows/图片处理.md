# 图片处理

AI 生成封面、信息图、文章配图并上传微信素材库。

## 步骤

1. 发现 provider: `wechat-pub providers list --json`
2. 发现 prompt 模板: `wechat-pub prompts list --kind image --json`
3. 生成封面: `wechat-pub generate_cover --article <article.md>`
4. 生成信息图: `wechat-pub generate_infographic --article <article.md> --preset <name>`
5. 上传单图: `wechat-pub upload_image <file>`
